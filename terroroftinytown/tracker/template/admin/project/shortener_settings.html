Alphabet: How to turn the sequence numbers into letters. Some common examples:

  • 01234567890abcdef
  • 0123456789abcdefghijklmnopqrstuvwxyz
  • 0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
  • 0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ_
  • 0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ-_

The boxes with HTTP status codes are whitespace seperated lists.

A redirect status code means there is a unshortened URL. A no redirect status code means that there is no shortened URL. A unavailable status code means that the shortener has content issues (like a DCMA notice) with unshortened URL. Banned status codes means that the scraper client has been banned.

Body regex is a space for a Python regex if the shortener only serves the unshortened URL in the HTML.

Tick the "custom code required box" if these settings are not adequate and additional logic is required in the client code.
